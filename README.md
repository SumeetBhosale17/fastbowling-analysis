# Fast Bowling Analysis
We will be analysis fast-bowling biomechanics and key attractors of fast bowling. And provide a feedback system which will provide feeback based on the analysis.

The key attractors:
- Back foot contact
- Brace front leg
- Non-bowling arm
- Lateral flexion
- Run-up
- Hip-Sholder seperation
- Jump and gather

For v1:
- Back Foot Contact
- Brace front leg
- Non-bowling arm (peak elbow height relative to sholder)
